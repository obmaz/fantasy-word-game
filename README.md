# Fantasy Word Game

## Overview
The Fantasy Word Game is a mobile-friendly, belt-scroll word guessing game that challenges players to match words in a fun and engaging fantasy-themed environment. This project includes a responsive design to ensure a seamless experience across various mobile devices.

## Project Structure
```
fantasy-word-game
├── src
│   ├── index.html        # Main HTML document for the game
│   ├── styles
│   │   └── style.css     # Styles for the game
│   ├── scripts
│   │   └── app.js        # JavaScript code for game logic
├── README.md             # Project documentation
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)
- Basic understanding of HTML, CSS, and JavaScript

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd fantasy-word-game
   ```

### Running the Game
1. Open `src/index.html` in your web browser.
2. Enjoy the game!

## How to Play
1. **Select a Day and Difficulty:**
   - Choose a "Day" to focus on a specific set of words.
   - Select the difficulty by choosing the number of monsters you want to face.
2. **Start the Battle:**
   - Click the "전투 시작" (Start Battle) button to begin the game.
3. **Answer the Questions:**
   - For normal monsters, you will be given a word or a meaning and you have to choose the correct translation from the given options.
   - For boss monsters, you will be given a meaning and you have to type the correct word in the input field.
4. **Use Skills:**
   - You can use the "힌트" (Hint) skill to get a hint for the current question.
   - You can use the "필살기" (Ultimate) skill to automatically answer the current question.
5. **Defeat all the monsters to win the game!**

## Boss Battles
- At least 20% of the questions in each game are boss battles, with a minimum of one boss battle per game.
- Boss battles are subjective questions where you have to type the answer instead of choosing from multiple options.
- Defeating a boss will give you a higher reward than defeating a normal monster.
