# 판타지 영단어 게임 (킹왕짱 RPG)

## 개요

판타지 영단어 게임은 플레이어가 재미있고 흥미로운 판타지 테마 환경에서 플레이할 수 있는 모바일 친화적인 영단어 맞추기 RPG 게임입니다. 이 프로젝트는 모바일 기기에 최적화된 9:16 비율의 반응형 디자인을 포함하고 있습니다.

## 시작하기

### 요구 사항

- 최신 웹 브라우저 (Chrome, Firefox, Safari 등)
- HTML, CSS, JavaScript에 대한 기본 이해

### 설치

1. 저장소 클론:
    ```bash
    git clone <repository-url>
    ```
2. 프로젝트 디렉토리로 이동:
    ```bash
    cd fantasy-word-game
    ```

### 게임 실행

1. 웹 브라우저에서 `index.html`을 엽니다.
2. 타이틀 화면과 함께 게임이 시작됩니다.
3. 드론다운에서 게임 데이터 세트를 선택합니다.
4. 게임 모드를 선택하고 게임을 즐기세요!

## 기능

### 다양한 게임 모드

- **연습 모드 (Practice Mode)**: 선택한 Day와 난이도로 연습할 수 있습니다.
- **배틀 모드 (Battle Mode)**: 모든 Day의 적들이 무작위 순서로 등장합니다.
- **보스 모드 (Boss Mode / 무한의 전장)**: 높은 점수를 위해 끝없이 진행되는 보스전입니다.

### 게임 기능

- 영단어-뜻(한글) 번역을 기반으로 한 단어 맞추기 게임 플레이
- 다양한 문제 유형 지원 (객관식, 주관식, 혼합형)
- 스킬 시스템 (힌트, 필살기)
- 골드 및 인벤토리 관리
- 아이템을 구매할 수 있는 상점 시스템
- 내구도가 있는 장비 시스템
- 통계 추적 기능

### UI 기능

- 모든 화면 크기에서 일관된 비율을 유지하며 모바일 기기에 최적화된 반응형 디자인
- 정밀한 버튼 매핑이 포함된 이미지 기반 UI
- 실시간 골드 및 타이머 표시
- 피해를 입을 시 모바일 기기 진동 피드백
- 루프 재생이 가능한 배경 음악
- 올바른 히스토리 관리를 통한 뒤로 가기 버튼 내비게이션 지원
- 다양한 게임 데이터 세트 지원
- 모달 및 화면에 대한 백드롭 블러 효과 적용
- 모바일과 데스크톱 모두에서 동일한 비율로 일관된 글꼴 크기를 유지하기 위한 clamp() 사용

## 플레이 방법

### 게임 시작 방법

1. **게임 데이터 세트 선택:**
    - 타이틀 화면의 드롭다운에서 선택합니다.
    - 선택 가능한 세트: "능률 보카 중등 기본" 또는 "워드 마스터 중등 고난도"

2. **게임 모드 선택:**
    - **연습 모드**: 특정 Day와 난이도를 선택합니다.
    - **배틀 모드**: 문제 유형과 난이도를 설정합니다.
    - **보스 모드**: 끝없이 진행되는 보스전을 즉시 시작합니다.

3. **모험 설정 구성 (연습/배틀 모드):**
    - "모험 지역" (플레이할 Day/스토리) 선택
    - "난이도" (몬스터 등장 수: 5, 10, 또는 20명) 선택
    - **배틀 모드 전용**: 문제 유형 (객관식/혼합형/주관식) 선택
    - "시작하기"를 클릭하여 게임 시작

4. **게임 플레이:**
    - **일반 몬스터**: 단어나 뜻이 주어지면 객관식 보기 중 알맞은 번역을 선택해야 합니다.
    - **보스 몬스터**: 뜻이 주어지면 입력란에 올바른 단어를 타이핑해야 합니다.
    - 정답을 맞히면 몬스터를 처치하고 골드를 얻어냅니다.
    - 오답 시 데미지를 입습니다 (장비의 내구도가 감소합니다).

5. **스킬:**
    - **힌트 (Hint)**: 현재 문제에 대한 힌트를 얻습니다.
    - **필살기 (Ultimate)**: 현재 문제의 정답을 자동으로 맞춥니다.

6. **상점 및 인벤토리:**
    - 상점을 방문하여 획득한 골드로 아이템을 구매할 수 있습니다.
    - 인벤토리를 관리하고 아이템을 장착하세요.
    - 아이템에는 사용 시 감소하는 내구도가 존재합니다.

7. **통계:**
    - 푼 문제 수, 정답 수, 정답률 등 게임 통계 기록을 확인할 수 있습니다.

8. **승리 조건:**
    - 선택한 난이도 내의 모든 몬스터를 처치하면 승리합니다.
    - 보스 전투는 일반 몬스터보다 더 높은 보상을 제공합니다.

## 게임 모드 상세

### 연습 모드 (Practice Mode)

- 특정 Day와 난이도를 골라 집중적으로 연습합니다.
- 카탈로그에서 원하는 Day를 모두 선택할 수 있습니다.
- 난이도 설정 (몬스터 5명, 10명, 20명)
- 특정 단어 집합에 대한 집중 학습에 적합합니다.

### 배틀 모드 (Battle Mode)

- 무작위 순서로 모든 Day에 등장하는 적들을 상대하게 됩니다.
- 그동안 학습한 모든 단어에 대한 지식을 테스트합니다.
- 문제 유형 설정 가능:
    - **객관식 (Objective)**: 다지선다형 문제만 출제됩니다.
    - **혼합형 (Mixed)**: 객관식과 주관식(타이핑) 문제가 50%씩 번갈아가며 출제됩니다.
    - **주관식 (Subjective)**: 주관식 타이핑 문제만 출제되며 모두 보스전으로 처리됩니다.
- 다음에 어떤 문제가 기다리고 있을지 알 수 없어 더욱 도전적입니다.

### 보스 모드 (Boss Mode / 무한의 전장)

- 끊임없이 이어지는 보스전입니다.
- 모든 문제는 보스 타입(주관식 타이핑 입력 필요)으로 제공됩니다.
- 최고 기록을 놓고 경쟁할 수 있습니다.
- 몇 명의 보스를 물리칠 수 있는지 한계가 없습니다.

## 보스 전투 상세

- **연습 모드**: 문제 중 최소 20%는 보스전으로, 게임당 최소 한 번은 보스와 반드시 전투하게 됩니다.
- **배틀 모드**: 보스 전투 등장 비율은 설정한 문제 유형에 따라 달라집니다.
    - 객관식 모드: 보스전 없음 (모든 문제가 다지선다형)
    - 혼합형 모드: 보스전 50% 비율 등장 (객관식과 번갈아 진행)
    - 주관식 모드: 모든 문제가 보스전
- 보스전은 여러 옵션 중 하나를 선택하는 대신 정답을 직접 타이핑해야 합니다.
- 일반 몬스터보다 보스 몬스터를 상대할 때 더 높은 골드 보상을 제공합니다.
- 보스 모드에서는 모든 질문이 보스 전투로 진행됩니다.

## 기술적 세부 정보

### CSS 아키텍처

- **모듈형 CSS**: 스타일을 여러 파일로 분리하여 관리를 용이하게 구성했습니다.
- **모드별 스타일**: 각 게임 모드마다 모달 및 화면에 사용되는 고유한 CSS 파일을 지니고 있습니다.
- **공통 스타일**: `modal-common.css`에서 공유 스타일을 관리합니다.
- **CSS 변수**: `variables.css` 파일에서 전역 변수를 정의합니다.
- **반응형 타이포그래피**: `clamp()`를 사용하여 모든 기기에서 글꼴 크기가 일관되게 조정되도록 설계되었습니다.
    - 기본 글꼴: `clamp(16px, 4vw, 24px)` - 뷰포트 너비에 비례해 확장됩니다.
    - 모든 UI 요소는 뷰포트 상대 단위(vw)를 사용하여 화면 비율을 일관되게 유지합니다.
    - 데스크톱 디스플레이에서도 모바일 레이아웃과 동일하게 배율로 확장되어 표시됩니다.
- **백드롭 필터**: 모달과 화면에 블러 효과를 적용해 시각적인 깊이감을 나타냈습니다.
- **Z-index 관리**: 적절한 레이어링 시스템 (타이틀 100, 모달 200, 결과 화면 300)을 사용합니다.

### 이미지 기반 UI

- 게임은 정밀하게 매핑된 버튼이 포함된 이미지 기반의 UI 요소를 사용합니다.
- CSS 변수를 활용하여 배경 이미지를 기준으로 버튼 위치를 계산했습니다.
- 각 게임 모드에 맞춰 다양한 레이아웃 스타일을 지원합니다.
- 최적의 성능을 위해 모든 이미지는 WebP 형식을 사용합니다.
- 이미지 크기에 기반한 동적 버튼 오버레이 동기화를 지원합니다.

### 데이터 관리

- **다중 데이터 세트 관리**: 게임 데이터 팩토리(game-data-1.js, game-data-2.js) 전환을 지원합니다.
- **동적 로딩**: 데이터 로더가 데이터 세트 간의 전환을 수행하고 관리합니다.
- **유사 단어 조합 (Decoy Words)**: 객관식 보기의 오답 스크립트가 동적으로 생성됩니다.
- **스토리 데이터**: 각 Day마다 고유한 스토리 콘텐츠(타이틀, 인트로, 승리 메시지)가 존재합니다.

### 모바일 최적화

- 터치 컨트롤이 포함된 모바일 기기에 맞게 최적화되었습니다.
- 피해 시 진동 피드백 제공 (모바일 기기 한정)
- `clamp()`를 이용한 일관성을 갖춘 모든 크기 반응형 디자인 지원.
- 데스크톱 디스플레이에서도 모바일 레이아웃과 동일하게 배율로 확장되어 표시됩니다.
- 백그라운드 음악 자동 재생 지원.
- 올바른 히스토리 상태 관리를 통한 뒤로가기 버튼 탐색 지원.
- 시각적인 깊이감을 추가하기 위한 백드롭 블러 효과 지원.

### 데이터 저장 스토리지

- 게임 진행 상황은 로컬 스토리지(`localStorage`)에 저장됩니다.
- 여러 세션에 걸쳐 골드, 인벤토리, 장비, 통계가 지속적으로 유지됩니다.
- 마지막으로 선택한 Day, 난이도, 문제 유형 정보가 기록됩니다.
- 로드할 때 이전의 선택된 데이터 세트가 그대로 복구됩니다.

- `shop-modal.css`: 상점 모달 스타일

- `statistics-modal.css`: 통계 모달 테이블 스타일

- `data-loader.js`: 데이터 세트 로딩 관리 및 변경 지원

### Data

- `game-data-1.js`: 게임 첫 번째 단어 세트 (능률 보카 중등 기본)
- `game-data-2.js`: 게임 두 번째 단어 세트 (워드 마스터 중등 고난도)
- `game-data-3.js`: 게임 세 번째 단어 세트 (추가 단어장)
- `items-data.js`: 아이템 설정, 상점 장착 아이템 기초정보 데이터베이스
- `decoy-words-set.js`: 객관식 보기에 사용되는 오답(decoy) 단어 풀
- `background_music_*.mp3`: 게임 배경 음악 파일들
